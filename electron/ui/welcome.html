快速开始

1

启动应用

运行 npm run dev — 编译 TypeScript 并启动 Electron 窗口，同时在 3399 端口启动 MCP 服务。

2

连接 MCP 服务

向 http://127.0.0.1:3399/mcp 发送 JSON-RPC initialize 请求以创建会话。后续所有工具调用需携带返回的会话 ID。

3

调用工具

使用 navigate 打开页面，execute_task 执行 AI 驱动的操作，screenshot 截取结果。组合使用可完成端到端测试流程。

4

运行测试套件

执行 npx ts-node tests/run-tests.ts 可自动运行所有 JSON 定义的测试套件。

可用 MCP 工具

navigate
导航浏览器到指定 URL，等待页面加载完成并重新注入 page-agent。
{ url: string }
execute_task
通过 page-agent AI 执行自然语言任务，LLM 会解析指令并执行 DOM 操作。
{ task: string }
get_page_state
获取当前页面 DOM 状态，包括简化的 HTML 结构和可交互元素索引。
{}
screenshot
截取当前视口的 PNG 截图并保存到磁盘。
{ path?: string }
click_element
通过 page-agent 元素树中的索引点击页面元素。
{ index: number }
input_text
向指定索引的输入框中输入文本内容。
{ index: number, text: string }
scroll
按指定方向（上/下/左/右）滚动页面，可设置滚动的视口页数。
{ direction: string, pages?: number }
get_status
查询 page-agent 当前的执行状态。
{}
stop_task
中止当前正在运行的 page-agent 任务。
{}

示例：MCP 工具调用

// 1. 初始化会话 POST http://127.0.0.1:3399/mcp { "jsonrpc": "2.0", "id": 1, "method": "initialize", "params": { "protocolVersion": "2025-03-26", "capabilities": {}, "clientInfo": { "name": "my-client", "version": "1.0.0" } } } // 2. 导航到目标页面 POST http://127.0.0.1:3399/mcp Headers: { Mcp-Session-Id: <会话ID> } { "jsonrpc": "2.0", "id": 2, "method": "tools/call", "params": { "name": "navigate", "arguments": { "url": "https://example.com" } } } // 3. 执行 AI 驱动的任务 { "method": "tools/call", "params": { "name": "execute_task", "arguments": { "task": "点击登录按钮" } } }
