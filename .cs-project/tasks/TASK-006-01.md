# TASK-006-01: CDP Client 基础层

## 父任务
TASK-006

## 涉及文件
- electron/core/cdp-client.ts (新增)
- electron/core/ipc-handlers.ts (修改：新增 executeCdp)
- electron/mcp/server.ts (修改：注册 execute_cdp)

## 方案要点
- 创建 CDP Client 模块，封装 Electron webContents.debugger
- 懒加载 attach：首次 CDP 调用时自动 attach debugger（1.3 协议版本）
- 页面导航时自动 re-attach（监听 did-navigate 事件）
- 应用退出时 detach
- 事件监听机制：支持注册 CDP 事件回调（供 network-interceptor 使用）
- 暴露 sendCommand(method, params) 和 on(event, callback) 接口
- ipc-handlers 新增 executeCdp 函数，直接调用 CDP Client
- MCP server 注册 execute_cdp 工具

## 验收标准
1. CDP Client 首次调用时自动 attach debugger
2. 页面导航后 debugger 自动恢复（不需要手动 re-attach）
3. execute_cdp MCP 工具能执行任意 CDP 命令并返回结果
4. CDP 事件监听机制可用（on/off）
5. 产生录制事件
