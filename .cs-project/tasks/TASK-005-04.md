## 方案

### Tab 切换
在搜索栏上方增加 Tab 切换：
- "项目录制" — 仅显示 scope=project 的录制
- "全局录制" — 仅显示 scope=global 的录制
- "全部" — 合并显示（默认）

### Tab UI
```html
<div class="scope-tabs">
  <button class="scope-tab active" data-scope="">全部</button>
  <button class="scope-tab" data-scope="project">项目录制</button>
  <button class="scope-tab" data-scope="global">全局录制</button>
</div>
```

### 录制卡片
- 每条录制显示 scope 标签（"项目" / "全局"），用不同颜色区分
- 项目录制: 蓝色标签
- 全局录制: 灰色标签

### 新建录制对话框
- 增加 scope 选择：项目级 / 全局
- 默认选 "项目级"

### 数据加载
```javascript
async function loadRecordings(searchQuery, scope) {
  // scope 传给 bridge.recorderStore.list(null, scope) 或 search(query, scope)
}
```

## 验收标准
1. 页面顶部有 Tab 切换（全部/项目录制/全局录制）
2. 切换 Tab 时按 scope 过滤展示
3. 每条录制卡片显示 scope 标签
4. 新建录制对话框可选保存位置（项目/全局）
5. 默认显示"全部"Tab，默认保存到"项目级"
