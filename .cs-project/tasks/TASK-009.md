# TASK-009: Electron 多实例池化管理

## 概述
支持多个 Electron 实例并发运行，通过实例池管理器统一调度，对外保持单一 MCP 端口。

## 验收标准
1. `MCP_PORT` 环境变量可覆盖默认端口 3399
2. `npm run start:pool` 启动实例管理器，对外监听 3399
3. 实例管理器自动启动 1 个 Electron（最少保留数），按需扩展到 CPU 核数
4. 请求到达时分配空闲实例，繁忙时等待或扩容
5. 空闲超过 5 分钟的实例自动关闭（保留至少 1 个）
6. 实例崩溃自动重启（保持最少 1 个）
7. Skill 无需改动，仍连接 `127.0.0.1:3399/mcp`

## 子任务
- TASK-009-01: Electron 适配（MCP 端口环境变量化）
- TASK-009-02: 实例池管理器 + 类型定义
- TASK-009-03: MCP 代理服务器 + 启动脚本

## 执行方案
串行：01 → 02 → 03
